# About This Challenge

Welcome to my **30 Days of OSINT (Open Source Intelligence) Challenge**.

Over the next 30 days, I'll be learning one OSINT topic every day, documenting my knowledge, practicing real-world concepts, and sharing my progress with the cybersecurity community.

This challenge is designed to strengthen my skills in **Open Source Intelligence**, **Threat Intelligence**, and **Cybersecurity Investigations** while building a public portfolio of my learning journey.

---

# 🎯 Challenge Goals

- Learn one OSINT topic every day
- Build strong investigation skills
- Practice using industry-standard OSINT tools
- Document my learning publicly
- Improve analytical thinking
- Stay consistent for 30 days
